## 🔨 What I'm building

### [EdgeMask](https://edgemask.pro) — LLM Security & Cost Gateway
> An API gateway that sits between your app and any LLM provider.
> Masks PII/PHI in real-time, cuts API costs ~40% via semantic caching.

- **DLP layer**: Detects and redacts sensitive data (SSNs, emails, PHI) before it reaches the model
- **Semantic cache**: Intent-aware caching — similar queries hit cache, not the API
- **Multi-model**: One endpoint for OpenAI, Anthropic, and Gemini
- **Zero-config**: Drop-in replacement — swap one `baseURL`, done

### [Scope](https://github.com/mr-isik) — AI Educational Assistant
> RAG-powered study assistant built for scale.

- Go backend · Docker · GCP Cloud Run
- Qdrant vector DB + OpenRouter for dynamic model routing
- Optimized for latency and cost at query time
